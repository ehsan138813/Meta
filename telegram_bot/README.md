# 🤖 ربات تلگرامی حرفه‌ای

یک ربات تلگرامی کامل با دکمه‌های شیشه‌ای (Inline Keyboard) و قابلیت‌های متنوع فارسی.

## ✨ قابلیت‌ها

- 🎲 **تاس بنداز** - تاس بنداز و شانستو امتحان کن
- 🎯 **شانس** - درصد شانست رو ببین
- 😂 **جوک** - ۷ جوک خنده‌دار فارسی
- 💎 **حقایق جالب** - حقایق علمی جالب
- 📅 **تاریخ و ساعت** - زمان فعلی
- 📊 **آمار** - تعداد پیام‌ها و اطلاعاتت
- ⚙️ **تنظیمات** - تغییر زبان و...
- 🔄 **دکمه‌های تعاملی** - منوی اصلی با دکمه‌های شیشه‌ای
- 💾 **ذخیره اطلاعات** - دیتابیس JSON

## 📦 نصب و راه‌اندازی

### مرحله ۱: نصب پایتون
اگه پایتون نداری از [python.org](https://python.org) دانلودش کن.

### مرحله ۲: دریافت توکن
۱. به [@BotFather](https://t.me/BotFather) در تلگرام برو
۲. دستور `/newbot` رو بفرست
۳. اسم رباتت رو انتخاب کن
۴. **توکن** رو کپی کن

### مرحله ۳: ویرایش فایل
فایل `bot.py` رو باز کن و این خط رو پیدا کن:
```python
TOKEN = "YOUR_BOT_TOKEN_HERE"
```
توکن خودت رو جایگزین کن.

### مرحله ۴: نصب کتابخانه‌ها
```bash
pip install -r requirements.txt
```

### مرحله ۵: اجرا
```bash
python bot.py
```

## 🎮 دستورات

- `/start` - شروع و نمایش منو
- `/help` - راهنما
- `/menu` - نمایش منو
- `/id` - دریافت آیدی عددی
- `/echo [متن]` - تکرار پیام

## 📁 ساختار فایل‌ها

```
telegram_bot/
├── bot.py           # فایل اصلی ربات
├── requirements.txt # کتابخانه‌های مورد نیاز
├── README.md        # این فایل
└── users_data.json  # دیتابیس (خودکار ساخته میشه)
```

## 💡 نکات

- ربات از دکمه‌های شیشه‌ای (Inline) استفاده می‌کنه که زیر پیام‌ها نمایش داده میشن
- اطلاعات کاربران در فایل `users_data.json` ذخیره میشه
- منوی فارسی کاملاً راست‌چین و زیبا
